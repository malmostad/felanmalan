Informations pour des utilisateurs PC/Windows

 

Cher client,

nous vous remercions d’avoir opté en faveur de notre produit.

Pour travailler avec les polices de caractères faites attention au suivant:

Système indispensable:
Microsoft Windows 98/NT 4.0 ou plus récent pour PostScript ou TrueType
Microsoft Windows 2000/XP ou plus récent pour OpenType

Afin de pouvoir travailler avec les polices PostScript sous Windows 98/Me/NT 4.0 sur votre ordinateur, il vous faudra le programme « ATM Light ». L’installation et l’exploitation de ce programme sont gratuites. Ce programme se trouve dans le dossier « Software/ATM Light/French » ou sur le site Web « http://www.linotype.com/freedownload » pour les télécharger gratuitement.

Pour des informations sur l’installation et l’administration des polices de caractères veuillez vous reporter au fichier « Documents/Manuals/Fonts_Administration/index.htm » ou sur le site Web « http://www.linotype.com/freedownload » pour les télécharger gratuitement..

Lisez le contrat d’utilisation du logiciel de polices (EULA) dans le dossier « Documents/License » ou sur le site Web « http://www.linotype.com/freedownload », s’il vous plaît.

Pour des informations intéressantes sur quelques polices de caractères (par ex. des plans du clavier) veuillez vous reporter au dossier « Documents/Font related ».

Nous vous remercions de votre attention.
